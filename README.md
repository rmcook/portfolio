# rmcook.github.io
Portfolio Website

## Installation

- Install Node.js

- Install dependencies by running `npm install`

- Install parcel bundler with `npm install -g parcel-bundler`

## Development

- To run the development server run `npm run start`

## Deployment

- To deploy run `npm run deploy`
